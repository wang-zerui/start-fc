# 阿里云函数计算：FC案例

## 本地快速体验

通过该应用，您可以简单快速的创建一个FC案例到阿里云函数计算服务。

- 下载命令行工具：`npm install -g @serverless-devs/s`
- 初始化一个模版项目：`s init devsapp/custom-golang-http-python3`
- 进入项目后部署项目：`cd custom-golang && s deploy`
